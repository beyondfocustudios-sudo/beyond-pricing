# OpenClaw System Map

## Módulos

### 1. Auth & Sessions
- **Funcionalidades**: Login (equipa e portal), OTP, Google e Microsoft OAuth.
- **Tabelas e Rotas**: Tabelas de usuários, sessões e permissões.

### 2. UI Shell & Design System
- **Funcionalidades**: Layout, componentes (cards, tabs), animações Base44.

### 3. Team /app
- **Funcionalidades**: CRUD de projetos, templates, checklists, insights e notificações.

### 4. Portal /portal
- **Funcionalidades**: Dashboard, visualizações de projetos, inbox, calendário.

### 5. Review & Approvals
- **Funcionalidades**: Comentários, threads, aprovações.

### 6. Dropbox
- **Funcionalidades**: Conexão, pré-visualização e download.

### 7. DB / RLS
- **Funcionalidades**: Tabelas, políticas, regras de exclusão.

### 8. QA / Hardening
- **Funcionalidades**: Testes Playwright e auditorias.
